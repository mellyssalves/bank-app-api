# Bank API

Projeto de um sistema bancário simples com:

- Java + Spring Boot
- API REST
- Transferência entre usuários
- Histórico de transações
- Frontend simples em HTML/JS

## Como rodar

1. Rodar backend (Spring Boot)
2. Abrir frontend (index.html)

## Funcionalidades

- Criar usuário
- Transferir saldo
- Ver usuários
- Ver histórico
